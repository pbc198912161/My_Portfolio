# DESCRIPTION OF TASK

## Overview
A responsive single-page personal portfolio website built using HTML, CSS, and JavaScript. The portfolio showcases my skills, projects, startup, experience, education, and contact information in a professional dark-mode design. The site is deployed live on Netlify.

## What I Performed
- Designed and built a complete professional portfolio from scratch using HTML, CSS, and vanilla JavaScript.
- Implemented a sticky navbar with smooth scroll, active link highlighting, and a hamburger menu for mobile.
- Built a hero section with a JavaScript typing effect that cycles through different roles automatically.
- Created an About section with animated stat boxes and a profile image with a circular purple border glow.
- Built a Skills section with 6 skill cards covering AI & Prompt Engineering, Cybersecurity, Security Tools, Technical & Cloud, Software & Design, and Creative skills.
- Added a Projects section showcasing 7 real projects with tech stack tags and GitHub links.
- Built a dedicated Startup section for ScrollSmart — my own startup — with a problem/solution layout, stat pills, and a link to the live landing page.
- Created a timeline-style Education & Experience section covering BCA, Anveshak Tech Club, ECLIF, Hackathon, and CODTECH Internship.
- Added an Interests section highlighting three published poetry books available on Google Play Books.
- Built a functional Contact section with a form and social links.
- Implemented scroll-triggered fade-in animations using IntersectionObserver.
- Made the entire site mobile-responsive with breakpoints at 900px, 768px, and 480px.
- Deployed the live site on Netlify.

## Features
- Sticky navbar with smooth scroll and active section highlighting
- JavaScript typing effect in hero section
- Scroll-triggered fade-in animations
- Skills grid with hover effects
- Projects grid with tech stack tags
- Startup section with ScrollSmart details
- Timeline-style experience section
- Contact form with success message
- Back-to-top button
- Fully responsive (mobile, tablet, desktop)
- Dark mode with purple gradient theme
- Live deployed on Netlify

## Technologies Used
- HTML5
- CSS3 (Flexbox, Grid, CSS Variables, Animations, Glassmorphism)
- Vanilla JavaScript (IntersectionObserver, DOM manipulation, Typing effect)
- Google Fonts (Poppins)
- Font Awesome Icons
- Netlify (Deployment)
